Course Planner — Binary Search Tree

CS-300 DSA: Analysis and Design (2025). C++ application using a Binary Search Tree to store and sort course data. Original implementation has no balancing, a missing delete function, and memory leaks.

Planned Enhancements
  • Add AVL tree self-balancing to guarantee O(log n) worst-case performance
  • Implement a proper delete() function with the three BST deletion cases
  • Add a destructor to eliminate memory leaks from unfreed tree nodes
  • Replace hardcoded course data with file input (CSV parsing)
  • Add prerequisite validation to detect circular dependencies
